Kurzanleitung zur Bildung:

Diese Partizipien werden von transitiven Verben im vollendeten Aspekt (СВ) gebildet:

  • -нн-: Bei Verben auf -ать / -ять (Vokal bleibt).
  • -енн- / -ённ-: Bei Verben auf -ить / -еть (Vokal entfällt).
  • -т-: Bei einsilbigen Stämmen oder Verben auf -нуть / -ыть.
1. Wähle das korrekte Suffix (-ать): прочитать → прочитаый
2. Wähle das korrekte Suffix (-ить): изучить → изучый
3. Suffix bei einsilbigen Verben: забыть → забыый
