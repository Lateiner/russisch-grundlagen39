Konsonantenwechsel & Suffixe

Regeln für Bildung & Wechsel:

Passive Partizipien Präteritum werden von SV-Verben gebildet. Achten Sie auf den Stammwechsel bei Verben auf -ить:

  • т → ч (встретить → встреченный)
  • п → пл (купить → купленный)
  • д → жд / ж (освободить → освобождённый)
1. Welches Suffix folgt auf -ать? сделать → сделаый
2. Welches Suffix folgt auf -ить? решить → решый
3. Suffix für „забыть“: забыть → забыый
4. Konsonantenwechsel т → ч: отметить → отмеенный
5. Konsonantenwechsel п → пл: купить → куенный
6. Zuordnung (Infinitive mischen):

Ziehen Sie die Wörter in die Suffix-Boxen. Die Auswahl wird bei jedem Reset gemischt!
